# Time Clicker 

Time Clicker is a clicker where you gain Time Units in order to advance the Timeline to discord new age

## Installation

Install Time Clicker with pip

```bash
  pip install -r requirements.txt
```
    
## Run


Start the server

```bash
  python3 TimeClicker.py
```
